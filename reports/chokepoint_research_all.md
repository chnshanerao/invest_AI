## EP.1 — CRDO（Credo Technology）| AI数据中心的"神经纤维"

### 一句话
AI数据中心里，GPU之间用铜缆连接。铜缆跑800G信号，5米就衰减成噪声。CRDO做的芯片嵌在铜缆两端，把噪声还原成信号。市占率73%。

### 关键数据
| 指标 | 数值 |
|------|------|
| 市值 | $462亿 (Mid) |
| TTM Revenue | $10.33亿 → FY2026全年$13.35亿 |
| 毛利率 | 68.5% (连续7期上升) |
| P/S | 44.8x |
| AEC市占率 | ~73% |
| 入场区间 | $120-145 |
| 当前价 | $250.81 |

### 产业链定位
```
GPU(NVIDIA) → HBM内存(MU/HYNIX) → SerDes连接(CRDO 73%) → 光互联(COHR/LITE) → 电力(LEU/SMR)
```

### 产品矩阵
| 产品线 | 功能 | 战略意义 |
|--------|------|----------|
| HiWire AEC | 铜缆内嵌SerDes芯片 | 核心现金牛，73%份额 |
| 光PAM4 DSP (ZeroFlap/Dove) | 光模块内DSP芯片 | FY2027光学>$600M |
| SerDes IP授权 | 卖IP给芯片设计方 | 生态渗透 |
| SerDes Chiplet | 异构集成芯片粒 | 对接chiplet趋势 |
| Line Card PHY/Retimer | 交换机信号调理 | 扩大TAM |

### 技术原理
SerDes = Serializer/Deserializer。将并行数据转串行高速传输，在铜缆两端做均衡补偿(EQ) + PAM4调制(4电平编码) + 前向纠错(FEC)。模拟电路用n-1成熟工艺，成本比先进制程低30-50%。

HBM内存从8层堆到16层，每多一层就多一道检测。CAMT做的检测设备，是先进封装产线的"质检员"。

### 关键数据
| 指标 | 数值 |
|------|------|
| 市值 | $89亿 (Small, Sweet Spot ✓) |
| TTM Revenue | $4.96亿 |
| 毛利率 | ~49% |
| CAGR | 22% |
| P/E | 222x trailing / ~55x forward |
| 入场区间 | $120-145 |
| 当前价 | $193 |

### 产业链定位
```
GPU设计(NVIDIA) → 晶圆代工(TSMC) → 先进封装(CoWoS/HBM) → 检测(CAMT) ← 每层都要检测
```

CAMT不是经典瓶颈——KLA是检测霸主(89.9%光学检测份额)。但CAMT在**先进封装检测细分**有独特性价比优势：比KLA便宜，专为OSAT后端设计。

模型局限：22% CAGR基于历史。HBM4(16层)量产=检测步骤翻倍→实际增速可能远超。

### 护城河 vs 风险
| 护城河 | 风险 |
|--------|------|

3. HBM4层数8→12→16，检测步骤翻倍

Sweet Spot ($8.9B)。先进封装检测TAM结构性增长。但当前$193高于入场区间$120-145，等回调。

## 附录: 方法论 — 如何复制这套研究

### 步骤1: 产业链定位 (5分钟)
找到公司在产业链的精确位置。上游是谁、下游是谁、能绕过去吗？

### 步骤2: 10-K精读 (1小时)
SEC EDGAR搜ticker → 最新10-K → 重点读:
- Item 1: Business → 产品线/客户/竞争对手/制造模式
- Revenue Concentration → 哪个客户>10%
- Risk Factors → 公司自己承认的威胁

### 步骤3: 瓶颈五问
1. 物理上不可跳过？
2. 只有1-2家供应商？
3. 下游有替代选择？
4. 产能能快速扩张？
5. 管理层在买自家股票？

### 步骤4: 财务建模 (30分钟)
从XBRL拿最近8季度Revenue/Gross Margin/Net Income → 算TTM/CAGR → 套Revenue Forward模型

### 步骤5: 竞争调研 (30分钟)
读竞争对手earnings call transcript → 搜索关键词确认竞争态势

### 步骤6: 内部人交易 (10分钟)
SEC EDGAR → Filing Type "4" → 只关注买入。卖出忽略(10b5-1常规)。

### 步骤7: 综合判断
瓶颈五问打分 + 估值模型 + 竞争格局 → BUY/HOLD/AVOID
